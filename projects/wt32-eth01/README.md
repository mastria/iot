### FT232RL -> WT32-ETH01 Connections

<img src="wiring.jpg" width="50%">

| **FT232RL Pin** | **WT32-ETH01 Pin** | **Description**     |
| -------------------- | ----------------------- | ------------------------- |
| VCC (5v)             | 5V                      | Power Supply              |
| RX                   | TX0                     |                           |
| TX                   | RX0                     |                           |
| GND                  | GND                     | Ground                    |
| GND                  | IO0                     | **On boot** to upload code|
